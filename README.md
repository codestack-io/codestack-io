# 👔 Clover Clothing

A modern full-stack e-commerce web application for purchasing **traditional men's clothing (Punjabi)**. Clover Clothing provides a seamless shopping experience with secure authentication, online payments, responsive design, and an intuitive user interface built with Next.js.

---

## 🚀 Live Demo

> **Live Website:** [**clover-clothing.vercel.app**](https://clover-clothing.vercel.app/)

---

## 📖 Overview

Clover Clothing is an online fashion platform specializing in traditional men's wear. Users can browse products, view detailed information, securely authenticate, add items to their cart, and complete purchases using Stripe. The application is built with modern web technologies to ensure excellent performance, security, and responsiveness across all devices.

---

## ✨ Features

* 🔐 Secure user authentication with NextAuth.js
* 👕 Browse and explore traditional Punjabi collections
* 🛒 Add products to cart
* 💳 Secure Stripe payment integration
* 📧 Email notifications using Nodemailer
* 📱 Fully responsive design for desktop, tablet, and mobile
* 🎨 Modern and interactive UI with animations
* ⚡ Fast server-side rendering and routing using Next.js
* 🔒 Password hashing using bcryptjs
* 🗄️ MongoDB database integration

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/codestack-io/clover-clothing.git
```

Move into the project directory:

```bash
cd clover-clothing
```

Install all dependencies:

```bash
npm install
```

```env
```

## ▶️ Running the Project

Start the development server:

```bash

```

Visit:

```text

```
